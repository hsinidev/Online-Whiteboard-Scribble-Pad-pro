An advanced, privacy-focused online whiteboard application built with modern web technologies. This project leverages the HTML5 Canvas API to provide a lag-free, native-feeling drawing experience directly in the browser.

Designed for education, remote collaboration, and creative brainstorming, providing a distraction-free galaxy environment for your ideas.

---

## ðŸŒŸ Key Features

*   **High-Performance Drawing Engine**: Direct manipulation of the HTML Canvas element with optimized React hooks for 60fps rendering.
*   **Zero Latency**: Instant response time, optimized for both mouse and touch inputs (mobile/tablet support).
*   **Privacy First**: 100% client-side execution. No data is sent to servers; your drawings stay in your browser's memory.
*   **Essential Tools**: 
    *   Dynamic Color Picker
    *   Adjustable Brush Size
    *   Undo History Stack
    *   One-click Clear
*   **Export Capability**: Instant download of your artwork as high-resolution PNG files with transparency.
*   **Immersive UI**: A custom-built, animated galaxy background system that adds depth without distracting from the canvas.
*   **SEO Optimized**: Built with extensive schema markup, meta tags, and semantic HTML structure.

## ðŸš€ Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/hsinidev/online-whiteboard.git
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

